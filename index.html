Frontend

Technológie:

  • HTML5, CSS3 - IE9 a vyššie
  • Vizualizácia dát - natívne grafy, infografika
  • Single page application
  • SOFEA - ďalší slajd
  • REST - JAX-RS
  • Tapestry - SS Component based, aj SOFEA
  • JavaScript MV* - vyššia abstrakcia pri implementácií - AngularJS

Návrh riešenia

Frontend - SOFEA

Návrh riešenia

Benefity vývoj

  • píše sa HTML, CSS - LESS. JS - Coffee script
  • Môže robiť web dizajner - non Java vývojar
  • FE plne v rukách web vývojára - optimalizácie, layout customizácia
  • Rýchla adopcia komponentov tretích strán
  • Hit refresh workflow - tvorím na živej aplikácií
  • Backend vývoj píše služby a vystavuje - jasný kontrakt

Návrh riešenia

Benefity prevadzka a používatelia

  • SPA šetrenie prenosového pásma
  • Klient môže fungovať offline
  • Verzionovať FE aj služby
  • Služby môže konzumovať aj iný FE (BE)
  • Štandardná technológia
  • Interaktivita

Návrh riešenia

Zápory

  • Duplicitná implementácia validácií
  • Prezradenie logiky a modelu na klienta
  • Vyťažovanie klienta
